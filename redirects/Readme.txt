This directory contains redirect rules, organized by top-level TOC buckets; one .json file per top-level TOC bucket.
For example, redirects that correspond to the DevTools top-level TOC bucket are covered in `.openpublishing.redirection.devtools.json`.

There are a few empty .json files, so that there's a redirect file for each top-level TOC bucket.

Instructions: https://review.learn.microsoft.com/help/platform/redirect-articles
