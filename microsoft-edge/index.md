# Microsoft Edge developer documentation

Microsoft Edge is the new browser built for Windows 10, providing fast, safe, and productive web experiences. The developer docs offer information on features and standards included in Microsoft Edge.  Learn about topics ranging from the latest supported APIs, building extensions, debugging with the F12 Dev Tools, and more.
