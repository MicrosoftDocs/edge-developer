# Reference \(WebView2\)  

The Microsoft Edge WebView2 control enables you to host web content in your application using [Microsoft Edge \(Chromium\)](https://www.microsoftedgeinsider.com) as the rendering engine.  See [WebView2](../../../webview2.md) for an overview and [Getting Started](../../GettingStarted.md) for a quick start.  [ICoreWebView2](0.9.488/ICoreWebView2.md) is a great place to start learning the details of the API.
