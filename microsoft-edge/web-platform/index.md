# AI for the web

Microsoft Edge supports the following:

_AI web APIs for sites_ are web APIs which you (a web app developer) can use from the JavaScript code in your website, to run AI models.

_Expose sites to agents_ are tools that you can use to give AI agents access to your website, such as to debug the website.
