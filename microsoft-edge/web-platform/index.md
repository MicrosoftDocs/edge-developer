# AI for the web

Microsoft Edge supports the following:

* **AI web APIs for sites** - Web APIs that you can use from the JavaScript code in your website, to run AI models.

* **DevTools for agents** - Tools that you can use to give AI agents access to your website, such as to debug the website.
