# JsValueRef Typedef

A reference to a JavaScript value.  
  
## Syntax  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## Remarks  
 A JavaScript value is one of the following types of values: Undefined, Null, Boolean, String, Number, or Object.  
  
## Requirements  
 **Header:** jsrt.h
