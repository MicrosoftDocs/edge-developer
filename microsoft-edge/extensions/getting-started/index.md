# Get started developing an extension
<!-- https://aka.ms/EdgeAddonsLearn -->

A basic extension for Microsoft Edge has a manifest file, usually named `manifest.json`.

To get started quickly, see [Sample: Picture viewer pop-up webpage](../samples/picture-viewer-popup-webpage.md).  Then you can copy, paste, and modify that sample directory.

For concepts, see [Extension concepts and architecture](./architecture.md).
